# Missiles over Xerion (MVP)

Minimal browser MVP for the "Missiles over Xerion" arcade-defense loop.

## Run

Use any static server from repository root, for example:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Controls

- `Mouse click`: launch interceptor (while playing)
- `Space` or `Enter`: start / restart
